\section{Limitations}

\begin{itemize}
  \item Synthetic-only training data may not match production code diversity.
  \item Tensor-network libraries may not support all 0.5B architectures equally.
  \item Quantum-inspired labels describe classical simulation---not quantum advantage claims.
  \item Decoherence regularization (exp\_010) is speculative (L4); null results expected.
  \item FreeMarker validation tooling is less mature than JavaScript/HTML parsers.
\end{itemize}
